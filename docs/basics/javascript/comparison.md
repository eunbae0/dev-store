# == vs ===

## 동등비교 (==)

- 암묵적 타입 변환을 통해 타입을 일치시킨 후 좌항과 우항을 비교한다.
- 타입이 일치하지 않아도 true로 반환한다.

```js
"6" == 6; // true
```

## 일치비교 (===)

- 좌항과 우항을 타입까지 일치하는지 확인하며 비교한다.
- 타입과 값이 모두 같아야 true로 반환한다.

```js
"6" === 6; // false
// highlight-next-line
"6" === "6"; // true
```

## 결론

동등비교는 예측 어려운 결과가 나오므로, 일치비교를 사용하는 것이 권장된다.
